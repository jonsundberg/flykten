# Content Package

Markdown-based content for the Flykten band website.

## Structure

- `press/` - Bio texts in three variants
- `releases/` - Release information and metadata
- `gigs/` - Gig information
- `contact.md` - Contact information

## Spotify för releases

För att visa Spotify-spelaren på hemsidan: i varje release under `links:` ska `spotify` vara en riktig albumlänk från Spotify (t.ex. `https://open.spotify.com/album/...`). Så här får du länken: öppna albumet i Spotify → klicka på … (tre prickar) → Dela → Kopiera länk till album. Ersätt placeholder-värdet `https://open.spotify.com/album/example` i `releases/revolutionen/release.md` med er länk.
